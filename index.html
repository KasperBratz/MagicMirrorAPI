MagicMirror

A program for MagicMirrors

View on GitHub Download .zip Download .tar.gz

Översikt över resultatet

MagicMirror är ett program för att man enkelt ska kunna sätta upp sin alledess egna magic mirror. Vad finns då för funktionalitet? Du kan bland annat visa reddit-posts, nyheter och vädret.

Tekniker

Programmet använder Jackson för att ta ut information ifrån Json-filer som är hämtade via diverse API.

Utvärdering

Vi har lyckats implementera de vitala delarna av våra ideér. Man kan visa nyheter, reddit-posts och se vädret. Det vi inte lyckades med var röststyrning ifrån IBM. Idén slopades eftersom tjänsten inte var gratis.

Projektgruppen

Projektgruppen består av Philip Pedersen, Kevin Ispiroglu Lindblad, Jakob Krantz och Kasper Bratz.
